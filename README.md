# Keyboard Switch

Switch between two keyboard. Base on https://github.com/xmdn/2_of_3_lang_switch_hotkey.

Authors: GLM-5🧙‍♂️, scillidan🤡.

Icon picture is form [SimpleKeys](https://beamedeighth.itch.io/simplekeys-animated-pixel-keyboard-keys) by [beamedeighth](https://beamedeighth.itch.io/).

## Usage:

1. Windows 10 → Advanced keyboard settings → Input language hot keys → Advanced key settings
	1. For example, Set `Alt+Shift+0` to switch to English
	2. Set `Alt+Shift+1` to switch to Chinese
2. `autohotkeyu64.exe KeyboardSwitch.ahk`
3. Type `Caps_Lock` to switch keyboard
